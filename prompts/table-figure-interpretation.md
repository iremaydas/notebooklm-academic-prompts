# Table and Figure Interpretation Prompt

Use this when an article has "difficult" tables, models, or figures.

```text
Help me interpret the tables and figures in this article.

For each important table or figure:

1. What is the purpose of the table or figure?
2. What variables or quantities are shown?
3. What is the main result?
4. How should I interpret the coefficients, effects, or patterns?
5. Which results are statistically or substantively important?
6. What does this table or figure contribute to the article’s argument?
7. Are there any results that are surprising, weak, or inconsistent?
8. Explain the table or figure in plain language.
```
