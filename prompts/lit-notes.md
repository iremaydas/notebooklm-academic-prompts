# Literature Review Notes Prompt

Use this when you want notes that can be helpful later. 

```text
Create literature review notes for this article.

Use the following format:

Citation:
[Author, year, title]

Research question:
[Write in one sentence]

Main argument:
[Write in 2–3 sentences]

Theory:
[Summarize the theoretical framework]

Data and methods:
[Summarize the data and method]

Findings:
[List the main findings]

Contribution:
[Explain how this article contributes to the literature]

Limitations:
[Identify important weaknesses or limits]
```
